# 🔧 Scripts

This directory contains standalone analysis scripts and utilities for the Barotrauma Model project.

## 📋 Contents

- **Data Processing**: Scripts for cleaning and preparing datasets
- **Batch Analysis**: Automated analysis pipelines
- **Model Validation**: Validation against experimental data
- **Report Generation**: Automated report creation

## 🏗️ Organization

```
scripts/
├── data_processing/        # Data cleaning and preparation
├── batch_analysis/         # Automated analysis pipelines  
├── validation/            # Model validation scripts
└── reporting/             # Report generation tools
```

## 🚀 Usage

Scripts are designed to be run independently:

```bash
python scripts/validation/validate_chamber_model.py
python scripts/analysis/batch_risk_assessment.py
```
