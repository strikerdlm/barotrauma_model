# Manuscript Repository Boundary

This repository is the executable model and software project for the middle-ear barotrauma simulator.

All manuscript submission work, rejection tracking, cover letters, journal-specific packages, figures prepared for upload, supplemental files, and publication strategy notes belong in:

- Local path: `/root/repos/manuscripts/barotrauma`
- Remote: `https://github.com/strikerdlm/manuscripts`

Do not commit manuscript packages, journal correspondence, rejection records, cover letters, submission checklists, or upload-ready DOCX/PDF/TIFF artifacts to this repository.

The only publication-related files that may remain here are lightweight project markdown files that describe the software, model assumptions, reproducibility contract, release process, or repository boundary. Substantive manuscript prose and submission history must live in the manuscripts repo.

When a manuscript needs model outputs, generate or validate them here, then copy only the derived manuscript artifacts into `/root/repos/manuscripts/barotrauma` and cite the exact model commit used.
